## 📌 О проекте

**Expense Tracker** — это десктопное приложение для контроля личных финансов. Программа помогает фиксировать ежедневные траты, анализировать расходы по категориям и периодам, а также хранить историю операций в JSON-формате.

### 🎯 Ключевые возможности

| Функция | Описание |
|---------|----------|
| ➕ Добавление расходов | Быстрый ввод суммы, категории и даты траты |
| 🗂 Категоризация | Готовые категории + возможность добавлять свои |
| 📅 Визуальный календарь | Выбор даты мышкой без ручного ввода |
| 🔍 Фильтрация | Поиск трат по категории и диапазону дат |
| 📊 Подсчёт суммы | Автоматический расчёт расходов за любой период |
| 🗑 Управление записями | Удаление ненужных строк из таблицы |
| 💾 Сохранение данных | Экспорт в JSON с автоматической загрузкой при старте |
